PC Engines WRAP GPIO control
----------------------------

Edit wrap_setup_gpio.c according to your needs,
unsigned char bit_cfg specifies whether pin is
used as input or output.

Requires 32-bit glibc-static.

Based on samples found on these pages:
 - http://wiki.voyage.hk/wrap_switches.txt
 - http://list.voyage.hk/pipermail/voyage-linux/2006-June/000969.html

WRAP datasheet:
 - http://www.pcengines.ch/pdf/wrap1c.pdf

SC1100 datasheet can be found on the internets.
Use at your own risk.
